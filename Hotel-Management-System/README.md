# SEPM I - Luxury Hotel Management System

**Project Initiation Date:** April 21, 2025

## Overview
**SEPM I** is a comprehensive, ultra-premium Hotel Management System designed as a capstone project for the Software Engineering Project Management (SEPM) module. The project focuses on delivering a flawless, minimalist User Interface (UI) engineered to provide a seamless booking and hospitality experience for luxury resorts.

The architecture emphasizes clean component separation, responsive design for high-class clientele across all devices (mobile, tablet, and desktop), and a visually striking aesthetic utilizing custom high-resolution imagery and robust CSS frameworks.

## Core Implementations
- **Resort Landing (index.html):** Features an immersive, full-screen hero gateway and a floating, dynamic booking engine.
- **Experience Hub (overview.html):** A polished display of the resort's premium amenities, from high-end suites to Michelin-grade dining.
- **Reservation Engine (book.html):** A robust, easy-to-navigate structural form ensuring frictionless room selection and requests.
- **Secure Auth Portal (login.html):** A sophisticated portal for guests and management alike.
- **Unified Style Architecture (style.css):** A modern, variable-driven CSS aesthetic focusing on the 'Champagne Gold' (`#D4AF37`) and Obsidian Dark/Light luxury palettes.

## Setup & Deployment
1. Clone the repository locally.
2. No live backend server is required for the UI display. Simply open `index.html` in any modern web browser to interact with the high-fidelity prototype.
